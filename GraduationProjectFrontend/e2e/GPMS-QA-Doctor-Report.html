1. Work Summary

Implemented

  • Added frontend unit/component testing with Vitest, Testing Library, Jest DOM, jsdom, and MSW-ready setup.
  • Added Playwright accessibility smoke tests using axe for landing, login, register, and authenticated dashboard shell.
  • Added performance smoke tests for public and authenticated routes.
  • Added backend auth schema unit coverage for registration, password, OTP, 2FA, and OAuth completion validation.
  • Improved Playwright config: HTML report, failure trace/screenshot/video, CI-only retries, Chromium default, mobile-only responsive specs.
  • Added a Windows-safe local Playwright runner that starts and stops backend/frontend explicitly.

Product fixes from tests

  • Fixed accessibility labels for app shell menu/search/account/notification buttons.
  • Fixed avatar accessible alt behavior and scroll-area keyboard focus.
  • Fixed landing/login/register/dashboard contrast issues found by axe.
  • Removed opacity-based page fades that made text temporarily fail contrast during automated accessibility scans.
  • Hardened supervisor-toolkit, GitHub provider, and performance smoke specs after real full-suite failures.

Files created

GraduationProjectBackend/src/modules/auth/auth.schema.test.js, GraduationProjectFrontend/vitest.config.ts, GraduationProjectFrontend/vitest.setup.ts, GraduationProjectFrontend/tests/unit/*, GraduationProjectFrontend/e2e/accessibility.spec.ts, GraduationProjectFrontend/e2e/performance-smoke.spec.ts, GraduationProjectFrontend/playwright.no-webserver.config.ts, GraduationProjectFrontend/scripts/run-playwright-a11y.mjs.

Files modified

package.json, package-lock.json, playwright.config.ts, e2e/README.md, landing/login/register/dashboard pages, app topbar, notification center, avatar, scroll-area, and stabilized GitHub/supervisor-toolkit E2E specs.
