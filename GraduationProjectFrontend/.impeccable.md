## Design Context

### Users
The primary users are university students, doctors, and teaching assistants (TAs) working on graduation projects. The interface needs to handle project management tasks (meetings, announcements, team coordination) in an academic environment that requires both professionalism and ease of use.

### Brand Personality
**Professional, Efficient, Modern**. The tone is academic yet forward-thinking. It should evoke confidence through structure and delight through smooth, purposeful interactions.

### Aesthetic Direction
Refined Glassmorphism. A clean, light-filled aesthetic using OKLCH color space for vibrant but professional tones. High emphasis on negative space, clear information hierarchy, and soft-shadowed containers. The theme supports both light and dark modes with a focus on legibility and reduced cognitive load.

### Design Principles
- **Clarity over Complexity**: Prioritize essential information; use progressive disclosure for details.
- **Visual Rhythm**: Use fluid spacing and consistent alignment to create a natural reading flow.
- **Intentional Motion**: Use Framer Motion for meaningful transitions that guide the user's attention.
- **Resilient Layouts**: Ensure the interface feels robust across all screen sizes using modern CSS techniques.
